# Emoji Reaction Picker

A modern Emoji Reaction Picker built using HTML, CSS, and JavaScript. This project features a sleek glassmorphism interface, smooth hover animations, persistent reaction storage, and intuitive interactions that make selecting an emoji fun and engaging.

Users can choose from multiple emoji reactions, instantly preview their selection, and have it automatically saved using Local Storage for future visits.

## Features

- Modern glassmorphism UI design
- Smooth emoji hover and pop animations
- Interactive emoji reaction picker
- Selected reaction preview
- Local Storage support to remember the last selected reaction
- Click outside or press **ESC** to close the picker
- Responsive and lightweight design
- Animated gradient background

## Tech Stack Used

- **HTML** – Structure of the Emoji Reaction Picker
- **CSS** – Glassmorphism styling, animations, responsive layout, and visual effects
- **JavaScript** – Emoji selection, Local Storage handling, picker interactions, and animations

## How to Run

1. Download the repository (A ZIP file will be downloaded).
2. Extract the ZIP file on your computer.
3. Open the extracted folder.
4. Double-click **index.html** or open it in any web browser.
5. Start selecting emoji reactions and enjoy the interactive experience.

**Note:** Make sure all files (`index.html`, `style.css`, and `script.js`) are kept in the same folder for the project to work correctly.
